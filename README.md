# 🌍 AirCast AQI Predictor

## 📌 Overview

AirCast AQI Predictor is a machine learning-based project that predicts the **Air Quality Index (AQI)** using environmental and pollution data.
It helps understand air quality trends and provides insights for better decision-making.

---

## 🚀 Features

* 📊 Data preprocessing and cleaning
* 🤖 Machine learning model training
* 🔮 AQI prediction system
* 🌐 Interactive web app using Streamlit

---

## 🛠️ Tech Stack

* **Python**
* **Pandas & NumPy**
* **Scikit-learn**
* **Streamlit**

---

aircast-aqi-predictor/

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Yogeshwaran16/Environmental-Monitoring-Hyper-Local-Air-Quality-Forecasting.git
cd Environmental-Monitoring-Hyper-Local-Air-Quality-Forecasting
```

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Run the Project

### Train the model

```
python src/train_model.py
```

### Run the Streamlit app

```
streamlit run app/app.py
```
